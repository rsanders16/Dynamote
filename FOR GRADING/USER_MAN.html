Fill out the form fields and press "Create Admin Account".

Managing User Accounts

Basic Authentication

Dynamote have one 'root' admin and allows an unlimited number of other 'Admin' and 'Standard' accounts to be added. All users, including the root admin will have to login via the login page. This was done for you automatically so let us go thorugh the process of logging in and out.

Logging Out

To logout, press 'Logout' (the rightmost hyperlink on the NavBar.

You should now see a screen similar to the following:

Side Note: The Dynamote Info Bar

Notice a green bar in the above screen shot. This bar informs you that you were successfully logged out. This green bar is called the Dynamote Info Bar. It is used to tell the user that things they did were carried out successfully or with errors (The bar would then be light pink)

Logging In

Log into Dynamote by clicking the login button. Enter the data you entered earlier to logging to Dynamote.

You should now see a screen similar to the following: (Notice the info bar says you were successfully logged in)

Also notice at the top of the screen "ADMINISTRATOR MODE". This text will only be preset if the logged in account is and admin.

Adding Accounts

Click on "Manage Accounts" in the NavBar

You should now see a screen similar to the following:

Add a new user by typing the new information in to the text fields at the bottom of the Account table printout (see below).

Adding an Admin Account

Click "Add":

To make the user an Admin, be sure to check the "Admin" checkbox (see below).

Adding accounts with the same username

Adding an account with the same username as another is not allowed. If you try Dynamote will stop you (see below).

Deleting Accounts

To delete and account, simply press the red x in the row corrisponding to the account you want to delete. Notice you can not delete the root admin account (The first account in the list). To do this you must reset Dynamote completely. Somthing you may not want to do.

Editing Your Account

To edit the account specific settings of your account. Press "Edit Your Account" on the NavBar. You should then see a screen like the one below:

Changing Your Password

Here you can update your password. Note that Dynamote checks to see if the old password is correct before allowing you to change the password to the new password.

Changing Your Locale

You can change the language of Dynamote to any of the supported languages. This change will only take effect for the logged in account and is not site wide. See below for a demo of the user changing the local to spanish

Changing the Theme Dynamote

You can change the theme of Dynamote to any of the theme. This change will only take effect for the logged in account and is not site wide. See below for a demo of the user changing the theme to rainbow

Dynamote Devices

It is now time to learn how to add devices so the Dynamote remote can do cool things with them

First click on "View All Devices" on the NavBar (see below).

The list is empty. Lets add a device:

Adding a Device

Dynamote needs the model number of the device so it can send the correct IR codes. Enter the model number of the device you want to use. If you add a device that Dyamote has detailed inforamtion on it will fill the rest of the form out for you. See below, when one adds the Model # CECHAxx, the name and type form fields are auto filled

Changing the Name of the device

Of course, if you do not like the name or type Dynamote pre-loads the form with, you can change it to your preference (see below)

Hiding and Un-hiding A Device

All users can hide devices so that they do not show up on their remote, all while still not having to uninstall the device form Dynamote completely. See the demo below:

Deleting a Device

To delete a device, simply press the red x in the row corresponding to the account you want to delete. NOTE: ONLY ADMINS CAN DELETE DEVICES. Dyanamote will block attempts to delete from other users.

Your Remote

After you have added devices to Dynamote its time to build and use your remote. Click on "Your Remote" on the NavBar. Start by draging buttons from the right onto the remote. You can change, add and delete rooms using the tabbed menu.
